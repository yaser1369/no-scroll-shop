# نمونه اپلیکیشن فروشگاهی بدون اسکرول

## اجرا
برای اجرای درست روی ویندوز:

1. پوشه را در VS Code باز کنید.
2. افزونه Live Server را نصب کنید.
3. روی `index.html` راست‌کلیک و `Open with Live Server` را بزنید.

## آدرس‌ها
- اپ مشتری: `index.html`
- پنل مدیریت: `admin.html`

## امکانات نمونه
- صفحه اصلی ثابت با نوار بالا و چهار مربع
- صفحه دسته‌بندی با چهار محصول در هر صفحه
- صفحه جزئیات محصول بدون اسکرول
- انتخاب رنگ، سایز و تعداد
- افزودن به سبد
- پنل مدیریت برای دسته‌ها و محصولات
- ذخیره اطلاعات در LocalStorage مرورگر
- PWA پایه

## محدودیت نسخه نمونه
این نسخه برای نمایش رابط کاربری و جریان کار است. اطلاعات در مرورگر ذخیره می‌شوند و هنوز به سرور، دیتابیس، ورود مدیر، پرداخت یا ارسال سفارش واقعی متصل نیست.
